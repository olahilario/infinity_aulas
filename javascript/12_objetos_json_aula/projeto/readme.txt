CATÁLOGO DE LIVROS ONLINE

Funcionalidades:

Listar Livros:
Criar uma função para exibir todos os livros disponíveis no catálogo.

Adicionar Livro:
Permitir aos usuários adicionar novos livros ao catálogo, incluindo informações como título, autor, gênero, ano de publicação e avaliações.

Buscar Livro:
Implementar uma funcionalidade de busca que permita aos usuários encontrar livros com base no título, autor ou gênero.

Filtrar Livros:
Adicionar a capacidade de filtrar os livros com base em diferentes critérios, como título, autor ou avaliação.

Avaliar Livro:
Permitir aos usuários atribuir avaliações aos livros existentes.
Salvar e Carregar do Arquivo JSON:

Implementar funções para salvar as alterações no arquivo JSON e carregar os dados existentes ao iniciar o programa.
Este sistema de catálogo de livros online oferecerá uma interface para os usuários visualizarem, adicionarem, buscaram, filtrarem e avaliarem livros. As informações dos livros serão armazenadas em um arquivo JSON para persistência de dados entre as sessões.
